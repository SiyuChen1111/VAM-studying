# Experimental checkpoint archive

This folder contains **one-off experimental checkpoint directories** that are no longer part of the primary read path.

These runs were kept for provenance, comparison, and possible future debugging, but they should not compete with the main checkpoint roots:

- `checkpoints_age_groups/`
- `checkpoints_age_groups_matched/`

Archived here are:
- diagnostic matrix runs
- objective-layer RT-shape runs
- response-agreement branch runs
- noise probes
- accumulator-RNN prototype runs

These experiments are documented in:
- `results/organized/handoff/logs.md`
- `results/organized/handoff/NEXT_STEP_DECISION_MEMO.md`

Use this directory only when you need historical artifact provenance.
